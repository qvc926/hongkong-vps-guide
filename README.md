## 先搞懂几个绕不开的术语：CN2、9929、CMIN2、BGP到底在说什么

买香港VPS之前，这几个词你必须认识，不然就是被人牵着鼻子走。

**CN2（AS4809）** 是中国电信的高端承载网，分CN2 GT和CN2 GIA两档。GIA是精品线路，延迟低、丢包少，是电信用户最理想的选择。简单理解：走CN2 GIA的香港VPS，电信用户访问体验会非常顺滑。

**9929（也叫CUII、AS9929）** 是中国联通的精品线路，比联通普通的169骨干网质量高出一截。联通用户走这条线回国，速度和稳定性都有保障。

**CMIN2（AS58807）** 是中国移动的精品优化线路，对应移动用户的回国需求。移动用户用普通国际线路经常体验拉胯，CMIN2就是专门解决这个痛点的。

**三网直连** 的意思就是：机房同时接入了电信、联通、移动三家的直连链路，不管你用哪家运营商，数据都能走对应的高端线路回国，不用绕道、不用中转。这才是真正意义上的"三网优化"。

**BGP优化** 则是机房通过BGP协议智能调度，根据用户来源自动选择最优路径。很多商家说的"BGP优化三网直连"，本质上就是用BGP把三家运营商的直连链路整合在一起，按来源分流。

> 一句话总结：真正的三网直连VPS，是让电信用户走CN2、联通用户走9929、移动用户走CMIN2，三家各走各的精品线路，互不打架。
